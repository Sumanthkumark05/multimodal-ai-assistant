## Project Description
Multimodal AI Assistant is an advanced AI application capable of understanding and processing multiple input types including text, images, and speech.

The project combines:
- NLP
- Computer Vision
- Speech Recognition
- Generative AI

to create a real-world intelligent assistant.

---

## AI Models Used
- GPT-2 for text generation
- Whisper for speech recognition
- BLIP for image captioning
